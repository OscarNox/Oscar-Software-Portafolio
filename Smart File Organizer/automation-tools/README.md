# Smart File Organizer

Herramienta de automatización desarrollada en Python que organiza archivos automáticamente en carpetas según su tipo.

## Problema

Las carpetas de descargas suelen llenarse de archivos mezclados, lo que dificulta encontrar documentos importantes.

## Solución

Este programa analiza cada archivo y lo mueve a una carpeta correspondiente:

- imágenes
- documentos
- videos
- archivos comprimidos

## Tecnologías

Python
OS
Shutil

## Uso

python organizer.py
